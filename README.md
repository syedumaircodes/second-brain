## 📖 About

I'm a software engineer who learns by writing things down. This repository is a public reflection of that habit — notes taken while reading books, completing courses, and working through ideas.

I write primarily for myself, but publish openly in case someone else is walking a similar path and finds something useful here.

> These are personal notes, not polished tutorials. Expect rough edges, honest confusion, and gradual clarity.

---

## 🗺️ How This Repo Works

This isn't structured like a course. Notes are added as I go — sometimes as bullet points, sometimes as full write-ups. The goal is consistency, not perfection.

If you find a mistake or want to discuss something, feel free to open an issue.
